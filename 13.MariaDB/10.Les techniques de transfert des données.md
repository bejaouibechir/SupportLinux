# Les tecniques de transfert des données

## **1. SQL natif + Stored Procedures (interne MariaDB)**

**Quand l’utiliser :** transformations simples, chargements réguliers.  
**Idée :**

- `LOAD DATA INFILE` pour charger rapidement des CSV.

- Procédures stockées pour nettoyer/transformer.

- Événements MariaDB pour automatiser.

**Avantage :** très rapide, aucune dépendance externe.  
**Limite :** pas adapté pour des pipelines ETL complexes.

---

## **2. Python (Pandas + SQLAlchemy)**

**Quand l’utiliser :** transformation lourde, logique métier, enrichissement.  
**Étapes typiques :**

- Extraction depuis fichiers API/CSV/Excel.

- Nettoyage via Pandas.

- Insertion en bulk dans MariaDB.

**Avantage :** flexibilité maximale.  
**Limite :** nécessite un runtime Python.

---

## **3. Outils ETL Open Source**

### **A. Apache Airflow**

- Orchestration complète d’un pipeline ETL.

- Connexions MariaDB prêtes à l’emploi.

### **B. Pentaho Data Integration (Kettle)**

- Interface graphique, facile pour les mappings.

### **C. Talend Open Studio**

- Très bon pour les flux complexes et connecteurs multiples.

**Avantage :** visuel, industrialisable.  
**Limite :** plus lourd à maintenir.

---

## **4. Outils de réplication / CDC pour MariaDB**

### **MariaDB MaxScale (CDC)**

- Capture des changements en temps réel (binlog).

- Idéal pour synchroniser MariaDB → Data Warehouse.

**Avantage :** quasi temps réel.  
**Limite :** installation MaxScale nécessaire.

---

## **5. Docker + scripts ETL automatisés**

- Un conteneur dédié ETL (Python, Bash, ou Airflow).

- Cron ou orchestrateur Kubernetes pour automatiser.

**Avantage :** portable, propre, versionnable.  
**Limite :** nécessite une infra container.

---

## **Conclusion brève**

| Besoin                     | Approche recommandée            |
| -------------------------- | ------------------------------- |
| Petits chargements rapides | `LOAD DATA INFILE` + procédures |
| Transformation lourde      | Python + Pandas                 |
| Pipelines industriels      | Airflow / Talend / Pentaho      |
| Temps réel                 | CDC via MaxScale                |
| Environnements modernes    | ETL en conteneur Docker         |
